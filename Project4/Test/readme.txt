contents of this zip file

this file contains test data and expected results for the recommender project
you are free to use these files to help you test your code before submitting to Gradescope.

PLEASE NOTE: just because you pass all of these tests does not mean you will pass all of the grade scope tests. BUT it will give you a good start.

this folder consists of the following:
- test data: files with book recommendation data, ratings.dat and small_ratings.dat
the format of these files is identical
each book recommendation takes 3 lines - user name, book name, rating (-5 to +5)

- sample input: .txt files - they contain commands for input to your program via the user interface

- sample output: .out files - the expected output using the sample input

- calculation walk thru.pdf: designed to help you think about the various aspects required to solve this problem
